📊 Evolución del Salario Real en Argentina (2019–2023)

Proyecto de Portfolio – Data Analyst
Agustín Ferrari

🎯 Pregunta de Negocio

¿Cómo evolucionó el salario real por sector económico en Argentina entre 2019 y 2023 y qué actividades lograron sostener o perder poder adquisitivo en un contexto de alta inflación?

📌 Hallazgos Principales

📉 Variación salarial real promedio (todas las actividades): -6%
→ Pérdida general de poder adquisitivo en el sector privado.

📈 Mejor desempeño:
CLAE 8 – Explotación de minas y canteras
→ +13,80% de variación salarial real media

📉 Peor desempeño:
CLAE 51 – Transporte aéreo
→ -23% de variación salarial real media

⚖️ Se detecta una fuerte heterogeneidad sectorial tras ajustar por inflación.

    ¿Qué demuestra este proyecto?

Limpieza y transformación de datos oficiales mensuales

Normalización de series temporales (base 2019)

Ajuste por IPC para cálculo de salario real

Ranking y análisis comparativo entre sectores

Extracción de insights con enfoque económico

Desarrollo de dashboard interactivo en Power BI
