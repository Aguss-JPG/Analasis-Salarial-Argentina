Análisis de la evolución salarial en Argentina (2019–2023)

Este proyecto analiza la evolución de los salarios medianos por actividad económica
(CLAE) en Argentina entre 2019 y 2023, utilizando técnicas de limpieza, EDA,
normalización y comparación relativa.

Dataset:
- Fuente: (acá después pondrías la fuente)
- Variables principales:
  - fecha
  - clae2 (actividad económica)
  - salario_median

Metodología:
- Limpieza de datos (eliminación de registros agregados y valores inválidos)
- Análisis exploratorio (distribuciones, outliers, tendencias)
- Normalización salarial con base 2019
- Ranking y percentiles de crecimiento por actividad

Hallazgos principales:
- La mayoría de las actividades muestran trayectorias similares post-2021.
- La actividad CLAE 64 presenta un crecimiento excepcional (+237,85%).
- CLAE 64 ocupa el puesto 1 del ranking y el percentil 100.

Prox:
- Ajuste por inflación (salario real)
- Comparación de salarios nominales
- Análisis de volatilidad por sector
